giantcellsim_master.py is the master file of the simulation. Options used to indicate the parameters used in the run.
Use --help option for more information on parameter options.
List of other necessary files:
giantcellsim_trial.py; runs a trial of the simulation
giantcellsim_motifoutput.py; runs simulations and controls motif data csv output
giantcellsim_allstrandoutput.py; controls all data csv output
giantcellsim_fulltrialoutput.py; controls full trial 1 data dsv output
giantcellsim_cell.py; defines Cell class
giantcellsim_population.py; defines Population class
